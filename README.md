# bitbank PnL Dashboard

bitbank APIを使った現物・信用取引の損益計算ダッシュボード。

## 機能

- **現物取引** の損益計算
- **信用取引（レバレッジ）** の損益計算
- **移動平均法 / 総平均法** を切り替えて計算
- **日次 / 月次 / 年次 / 全期間** で集計
- リアルタイム残高・時価表示
- 銘柄別損益内訳
- オープンポジション一覧

## セットアップ

```bash
npm install
npm run dev
```

## Vercelデプロイ

```bash
npm i -g vercel
vercel
```

## bitbank APIキーの設定

1. bitbankのアカウントにログイン
2. `設定 > API` からAPIキーを発行
3. 読み取り権限のみ付与（取引履歴・残高・ポジション）
4. ダッシュボードの「API設定」からAPIキーとシークレットを入力

## 注意

- APIキーは**読み取り専用**権限のみで十分です
- APIシークレットはVercelサーバーサイドで署名処理に使用されます
- 秘密鍵が漏洩しないよう取り扱いに注意してください
- このツールは取引を実行しません

## 損益計算方式

### 移動平均法
買付のたびに平均取得単価を更新します。
`新しい平均単価 = (既存残高 × 現在の平均単価 + 購入額) / 新しい残高`

### 総平均法
期間全体の購入合計から平均取得単価を算出します。
`平均取得単価 = 期間内の購入合計額 / 期間内の購入合計数量`
